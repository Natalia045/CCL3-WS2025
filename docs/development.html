App Concept

NoMooSugar encourages users to reduce sugar intake through fun, visual motivation and gamified experiences rather than strict rules.

Use Cases

  • Logging sugar intake of consumed food or drinks
  • Tracking daily sugar intake
  • Completing challenges and streaks
  • Getting motivated by visuals

Target Users

Demographics

  • Age: 18–30 y.o. (Digital Natives)
  • University students or young adults

Interests

  • Healthy lifestyle
  • Cute-core, soft aesthetic
  • Games and gamified experiences
  • Social media and online trends

Needs

  • Motivated by fun rather than strict rules
  • Visual progress, rewards, and challenges
  • Easy-to-use, low-effort tracking tools

Final Reflections

Natalia

This has been my favorite CCL so far because of the topic. The most challenging part was learning how to work together with someone in real time, which can be really frustrating in the coding world. For me, the hardest part was merging the code, but I think we learned a lot from that experience. I also gained insight into what makes a good mobile application, realized that sometimes I need to adjust my expectations to achieve our goals, and learned a lot about using GitHub and Android Studio at the same time. We made sure to split the work evenly. I contributed to every part, including design, coding, brainstorming, and documentation. Overall, I think we met our goals and created a fully functioning app with a great user experience. Of course, there’s always room for improvement. If we had more time, I would like to add new types of challenges, track consecutive days, implement a login for a more personalized experience, and create a more precise system for awarding points through challenges.
