Brief Description

NoMooSugar is an Android app that helps users reduce their sugar intake in a playful way. It uses a cow-themed design to gamify the experience. Users can track their daily sugar intake, set goals, and build healthier habits over time.

Use Cases

  • Logging sugar intake of consumed food or drinks
  • Tracking daily sugar intake
  • Completing challenges and streaks
  • Getting motivated by visuals

Target Users

Demographics

  • Age: 18-30 y.o. (Digital Natives)
  • University students or young adults

Interests

  • Healthy lifestyle
  • Cute-core, soft aesthetic
  • Games and gamified experiences
  • Social media, online trends

Needs

  • Motivated by fun rather than strict rules
  • Visual progress, rewards, and challenges
  • Easy-to-use, low-effort tracking tools
