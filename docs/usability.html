Usability Test Plan

The usability test aimed to evaluate ease of use, clarity of navigation, and user motivation while tracking sugar intake.

Heuristic Evaluation

A heuristic evaluation was conducted on the initial prototype to identify usability issues related to navigation, feedback, and visual clarity.

Hypotheses / Testable Questions

  • Users can easily log sugar intake without instructions.
  • The cow-themed visuals increase motivation.

Planned Data Collection

  • Task completion time
  • Error rate
  • User satisfaction ratings

User Test Results

The user test included multiple participants within the target age range. Feedback was collected through observation and questionnaires, highlighting strengths in visual appeal and areas for improvement in data entry speed.

Updates Based on Feedback

Based on the feedback, UI elements were simplified and input steps were reduced to improve efficiency and overall user satisfaction.
